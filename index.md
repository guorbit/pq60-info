# PQ60 Standard

An electrical standard for PocketQubes

{: .highlight }
This website is brought to you by GU Orbit, University of Glasgow's student CubeSat society! We are implementers of this standard.

## About

PocketQubes (PQs) are 5cm×5cm×5cm size pico-satellites. The working principles behind PQs are similar to any other class of satellite. For this reason the potential of this class of satellite as a teaching and design aid is significant.

This site represents an electrical standard, the PQ60 Standard. This standard describes a common electrical interface between boards within a PQ structure. It is designed to complement the PQ mechanical standard.
